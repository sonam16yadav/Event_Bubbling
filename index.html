EventBubbling

child

In JavaScript, when an event (like a click or keydown) occurs on an element, it bubbles up from that element to its parent elements, eventually reaching the root of the document. This is known as event bubbling.

The event first triggers on the target element (the one you interact with, e.g., a button).

The event then bubbles up to the parent element, and then to its parent, and so on, until it reaches the root (document or window).

Event bubbling allows us to attach a single event listener to a parent element, and handle events for all its children, which is called event delegation.
